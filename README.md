# 📊 Dashboard de Dados de Alunos

Este projeto é uma aplicação web desenvolvida em React que coleta dados de alunos através de formulários do Google Forms integrados ao Google Sheets e transforma essas informações em gráficos interativos.

O objetivo é facilitar a visualização e análise de dados acadêmicos de forma simples, rápida e visual.

---

## 🚀 Funcionalidades

- Integração com Google Sheets para coleta de dados em tempo real  
- Geração automática de gráficos interativos  
- Visualização de dados como:
  - Idade dos alunos  
  - Curso  
  - Semestre atual  
  - Situação profissional (estágio/trabalho)  
- Dashboard responsivo e moderno  
- Reorganização dinâmica dos gráficos na interface  
- Interface intuitiva e fácil de usar  

---

## 🛠️ Tecnologias utilizadas

- React.js  
- JavaScript (ES6+)  
- Recharts (gráficos)  
- Axios (requisições HTTP)  
- Google Sheets API  
- Bootstrap / CSS
